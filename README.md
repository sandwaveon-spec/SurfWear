# Tidewear

A brand and shop site for a **cold-water surf & coastal apparel** label. A bright, airy editorial lookbook
&mdash; Space Grotesk display type, a foam/sea/coral palette, big coastal photography, clean colour-swatch
product cards, and a genuine **basket-pricing** core.

## Pages
- **Home** &mdash; coastal hero, seasonal essentials, our stance
- **Shop** &mdash; the range as colour-swatch product cards
- **Lookbook** &mdash; the AW15 coastal campaign grid
- **Basket** &mdash; live subtotal, kit-up bundle discount, free-shipping threshold

## Stack
Static HTML/CSS/JS. The basket pricing (`assets/js/cart.js`) is verified in Node.

## Verify
```
node scripts/verify-core.mjs
```
